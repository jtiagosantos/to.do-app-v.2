## 🚀 Run Project

1️⃣ Clone project and access its folder:

```bash
$ git clone https://github.com/jtiagosantos/to.do-app.git
$ cd to.do-app
```

2️⃣ Install dependencies:

```bash
$ yarn or npm i
```

3️⃣ Start project:

```bash
//android
$ yarn android or npm run android

//ios
$ yarn ios or npm run ios
```

4️⃣ Run tests:

```bash
$ yarn test or npm run test
```
